# NBT
Simple and fast NBT library. Currently does not support compressed formats.

## Gradle
```groovy
repositories {
    maven {
        name = "meteor-maven"
        url = "https://maven.meteordev.org"
    }
}

dependencies {
    implementation "meteordevelopment:NBT:0.1.1"
}
```
